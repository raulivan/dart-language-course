# 🎯 Dart Language Course

Bem-vindo ao **Dart Language Course**! Este repositório foi criado com o objetivo de apresentar uma visão geral e aprofundada da linguagem de programação [Dart](https://dart.dev/), abordando desde os fundamentos essenciais até os recursos mais avançados.

Seja você um iniciante em programação, um desenvolvedor buscando aprender Flutter, ou alguém querendo dominar o Dart no back-end, este material foi pensado para estruturar o seu aprendizado passo a passo.

---

## 📚 Estrutura do Curso (O que você vai aprender)

O repositório está dividido em módulos lógicos para facilitar a sua jornada de aprendizado:

### 🟢 Módulo 1: Fundamentos
* Sintaxe Básica e Estrutura de um programa Dart (`main`)
* Variáveis e Tipos de Dados embutidos (`int`, `double`, `String`, `bool`)
* Null Safety básico (`?`, `!`, `??`)
* Operadores (Aritméticos, Relacionais, Lógicos)
* Estruturas de Controle de Fluxo (`if`/`else`, `switch`, `for`, `while`)

### 🟡 Módulo 2: Funções e Coleções
* Funções (Parâmetros nomeados, posicionais, opcionais e anônimas/lambdas)
* Coleções Básicas (`List`, `Set`, `Map`)
* Manipulação e métodos de coleções (`map`, `where`, `reduce`)

### 🟠 Módulo 3: Orientação a Objetos (POO)
* Classes, Objetos e Construtores
* Encapsulamento (Getters e Setters)
* Herança e Polimorfismo
* Classes Abstratas e Interfaces (`implements`)
* Mixins (`with`) - Uma das funcionalidades mais poderosas do Dart
* Extensions (Métodos de extensão)

### 🔴 Módulo 4: Programação Assíncrona
* Entendendo o Event Loop
* `Future` e o padrão `async` / `await`
* Tratamento de exceções assíncronas (`try` / `catch`)
* `Stream` e `StreamController` (Programação reativa básica)

### 🟣 Módulo 5: Recursos Avançados
* Generics (Tipos Genéricos)
* Null Safety Avançado (Late variables, Flow analysis)
* Isolates e Concorrência (Multithreading no Dart)
* Metaprogramação e Annotations
* Testes Unitários básicos com `test`
